# Replication Archive for: Predicting Results of Social Science Experiments Using Large Language Models

This repository contains the code and data necessary to replicate the results in our [paper](https://www.treatmenteffect.app/paper.pdf)

Results from a full reproducible run of this code will soon be made available on [Open Science Library](https://codeocean.com/explore?page=1&filter=all) (currently only accessible to peer reviewers)
